## ✨ Fitur Utama (Features)

Aplikasi **Manajemen Data Mahasiswa** berbasis mobile yang dibangun secara native menggunakan framework Flutter. Aplikasi ini dirancang untuk memberikan pengalaman pengguna yang mulus dalam mengelola data.

- 🔐 **Autentikasi:** Halaman Login & Register.
- 📋 **Daftar Mahasiswa:** Melihat semua data mahasiswa secara *real-time*.
- ➕ **Tambah Data:** Form pengisian data mahasiswa (kini dilengkapi dengan fitur *Date Picker* kalender interaktif).
- ✏️ **Edit Data:** Memperbarui data mahasiswa dengan mudah.
- 🗑️ **Hapus Data:** Menghapus data dengan konfirmasi yang aman.
- 📱 **Responsif & Elegan:** Desain modern menggunakan komponen Material Design dan indikator loading yang elegan.

## 🚀 Memulai (Getting Started)

Ikuti panduan berikut untuk menjalankan project ini di komputer lokal kamu.

### Persyaratan Sistem
- [Flutter SDK](https://flutter.dev/docs/get-started/install) (versi terbaru direkomendasikan)
- [Dart SDK](https://dart.dev/get-dart)
- Emulator Android / iOS Simulator / Perangkat Fisik (HP)

### Instalasi

1. **Clone repository ini**
   ```bash
   git clone https://github.com/Zeeinch/Flutter-App-mahasiswa-.git
   ```

2. **Masuk ke direktori project**
   ```bash
   cd Flutter-App-mahasiswa-
   ```

3. **Install semua dependencies / library**
   ```bash
   flutter pub get
   ```

4. **Jalankan aplikasi**
   ```bash
   flutter run
   ```

## 📂 Struktur Folder `lib/`

| File | Deskripsi |
|------|-----------|
| `main.dart` | Titik masuk utama aplikasi (Main Entry Point) dan konfigurasi *routing*. |
| `login_page.dart` | Antarmuka pengguna untuk halaman Login. |
| `register_page.dart` | Antarmuka pengguna untuk halaman pendaftaran akun baru. |
| `home_page.dart` | Halaman Dashboard utama setelah pengguna berhasil login. |
| `mahasiswa_list_page.dart` | Menampilkan seluruh data mahasiswa dalam bentuk *list*. |
| `mahasiswa_form_page.dart` | Form interaktif untuk menambahkan data mahasiswa baru. |
| `mahasiswa_edit_page.dart` | Form interaktif untuk mengubah data mahasiswa yang sudah ada. |
| `mahasiswa_detail_page.dart` | Halaman untuk melihat detail informasi satu mahasiswa secara lengkap. |
